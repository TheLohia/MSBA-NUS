model = make_pipeline(StandardScaler(), LogisticRegression(penalty = 'none'))

9. Compare and comment on the validation results using X, X_single and X_all. Do these agree with your observations in question 2? Which is the best model?¶

(3 marks)

Answer: When comparing the validation results using X_single to the validation results of either of the two other datasets, we see a stark difference in model performance, as is expected considering that no single feature is able to clearly distinguish chinstrap penguins from the rest. While the model does return a perfect precision score for 2 of the 5 folds, it performs quite poorly in all the other metrics for all the folds. Hence, we can safely say that the model using X_single is a poor model.

There is an interesting observation to be made when comparing the models trained using X and and X_all. While we would expect that more features leads to better model performance, we actually notice that for every metric and every fold, the model trained using X performs equally, or better than the model trained using X_all. Hence, based on this observation, we can conclude that the model trained using X is the best model based on the four metrics under consideration

Hence, I can conclude that the results match the observations made in question 2 as using a single feature leads to a poor model, and using the two features culmen_length_mm and body_mass_g yields the best model in terms of performance across the four metrics we consider. The drop in performance upon adding more features suggests that the addition creates more noise than signal, making it more difficult for the model to identify the decision boundary efficiently.
